Introduction
----------------
Welcome to the Beken BK7238 development guide documentation center! This documentation center is designed to assist developers in quickly developing smart IoT products based on the Beken chip platform. It aims to meet the user's requirements for Bluetooth LE, Wi-Fi, low power consumption, etc. This documentation center includes a quick start guide, introduction to APIs , and relevant application examples, etc.
